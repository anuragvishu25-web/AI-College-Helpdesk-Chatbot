# AI College Helpdesk Chatbot 

## Project Descroption 
This project is an AI-based chatbot devloped using python and Machine learning to answer college related queries such as admissions ,fees, and courses.

## Technologies used 
-python
-flask
-natural language processing 
-scikit learn 
-tf -idf vectorizer 

## how to run the project 
1.clone the repository 
2.install the requirments 
3.train the model 
4.run the flask app
